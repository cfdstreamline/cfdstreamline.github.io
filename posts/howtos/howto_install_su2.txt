Pre installations

1. Install swig
sudo yum install swig

2. Install MPICH

3. Install mpi4py

4. It is not needed to install Meson, Ninja, CoDiPack and MeDiPack. SU2 installation process does that for you and ensures the correct versions.

----------------------------------------------------------------------------

https://su2code.github.io/docs_v7/Build-SU2-Linux-MacOS/
https://su2code.github.io/docs_v7/SU2-Windows/


0. Make sure you are using bash.
echo $SHELL
# if it is not bash, command: bash


1. Create a folder for mpich
mkdir /home/you/su2

2. Download to that folder the latest stable source code version of su2 from:
https://su2code.github.io/download.html

3. Unzip the file
unzip su2code...zip


4. Create 2 other folders
mkdir /home/you/su2/build


5. Enter the unziped su2 source code folder and run mason
cd /home/you/su2/su2code...
./meson.py build -Denable-autodiff=true --prefix=/home/you/su2/build

6. Compile with ninja
./ninja -C build install
